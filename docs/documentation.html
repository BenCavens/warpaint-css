Grid gutter

By default there is no gutter between columns. By adding .gutter to your .row element, all child columns will receive a gutter in between.

Container

By default the grid expands to the full width of its parent. It can be useful to nest the grid inside a .container element. This way it will receive a nice left-right responsive margin, relative to its parent.

5
4
3
<div class="container">
    <div class="row">
        <div class="column-5">5</div>
        <div class="column-4">4</div>
        <div class="column">3</div>
    </div>
</div>

Spacing

We define two types of spacings: the spacing of elements inside their parent container (margins) and spacing inside the element (paddings).

.stack

Handle margins with .stack. Stack sets the top and bottom margin. Since the page elements are mostly positioned vertically, this class is used to add breathing room in between.

.stack

.inline

Inline class sets a horizontal margin. This can be used for adding space between sibling elements like pills, tags, breadcrumbs or buttons.

.inset

Paddings of an element can be managed by .inset, .stretched or .squished. Inset is the most common and sets a square padding: padding on four sides of the element.

.squished

A squished inset stretches horizontally. It reduces space top and bottom by 50% of the inset. This is mostly used for buttons and cell-like containers as tables or lists

.stretched

A stretched padding is the opposite of squished. It stretches vertically and is ideally for textboxes and other form elements
